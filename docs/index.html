Navbar

<div class="navbar collapsed">
    <a class="navbar-title" href="javascript:void(0)">ThodaCSS</a>
    <ul class="pull-right">
        <li><a href="javascript:void(0)">Link</a></li>
        <li><a href="javascript:void(0)">Link</a></li>
        <li><a href="javascript:void(0)">Link</a></li>
    </ul>
    <a href="javascript:void(0)" class="menu-toggle-button">&equiv;</a>
    </a>
</div>

If you want a fixed navbar, add class="fixed" to the navbar element:

<div class="navbar fixed">
...
</div>

For better mobile support, ThodaCSS provides a collapsible navbar (as you can see on the home page). To use the collapsible navbar, add class="collapsed" to the navbar and also include navbar.js: 

<div class="navbar collapsed">
    <a class="navbar-title" href="javascript:void(0)">Navbar Title</a>
    <ul class="pull-right">
        <li><a href="javascript:void(0)">Link</a></li>
        <li><a href="javascript:void(0)">Link</a></li>
        <li><a href="javascript:void(0)">Link</a></li>
    </ul>
    <a href="javascript:void(0)" class="menu-toggle-button">
        &equiv;
    </a><!-- To toggle the navbar -->
</div>

<script src="js/navbar.js"></script>

Hero Unit

A hero unit should be the immediate child of the body element. 

<div class="hero">
    <div class="hero-content">
        <h1>Page Title</h1>
        <p class="tagline">Some cool tagline.</p>
</div>
